# 背景图像预处理指南

## 概述

本指南说明如何为静态背景相减法准备高质量的纯净背景图像。

## 1. 拍摄纯净背景图像

### 1.1 环境准备
- 确保实验场地无任何前景物体（包括小鼠）
- 保持光照条件与实验时一致
- 移除所有可能移动的物体

### 1.2 拍摄要求
- 使用与实验相同的摄像头和分辨率
- 确保画面清晰、对焦准确
- 避免画面中出现人员或其他干扰

### 1.3 拍摄时机
- 在实验开始前拍摄
- 光照稳定时进行
- 建议拍摄多张备用

## 2. 背景图像编辑处理

### 2.1 推荐软件
- **Adobe Photoshop** - 专业级图像编辑
- **GIMP** - 免费开源替代方案
- **Paint.NET** - 轻量级选择

### 2.2 编辑步骤

#### 使用 Photoshop/GIMP 的仿制图章工具：

1. **打开背景图像**
   ```
   文件 → 打开 → 选择背景图像
   ```

2. **选择仿制图章工具**
   ```
   工具栏 → 仿制图章工具 (快捷键: S)
   ```

3. **设置画笔参数**
   - 画笔大小: 根据需要调整 (建议 20-50 像素)
   - 硬度: 0-30% (边缘柔和)
   - 不透明度: 80-100%

4. **清除干扰元素**
   - 按住 Alt 键点击干净的背景区域作为采样源
   - 在干扰元素上涂抹，用背景替换
   - 重复此过程直到所有干扰被清除

5. **检查处理结果**
   - 放大图像检查细节
   - 确保没有明显的修复痕迹
   - 边缘过渡自然

### 2.3 常见干扰元素处理

| 干扰类型 | 处理方法 |
|---------|---------|
| 小鼠残留 | 仿制图章工具覆盖 |
| 粪便痕迹 | 采样周围地板区域覆盖 |
| 反光点 | 降低不透明度处理 |
| 阴影 | 采样相邻区域平衡 |

## 3. 文件管理规范

### 3.1 文件格式
- **推荐格式**: PNG (无损压缩)
- **备选格式**: BMP, TIFF
- **避免格式**: JPG (有损压缩)

### 3.2 文件命名规范
```
background_[实验名称]_[日期]_[版本].png

示例:
background_openfield_20240115_v1.png
background_watermaze_20240115_v2.png
```

### 3.3 目录结构建议
```
ETHO/
├── backgrounds/
│   ├── openfield/
│   │   ├── background_20240115_v1.png
│   │   └── background_20240115_v2.png
│   ├── watermaze/
│   │   └── background_20240115_v1.png
│   └── ymaze/
│       └── background_20240115_v1.png
├── videos/
├── results/
└── static_background_tracker.py
```

## 4. 系统集成

### 4.1 使用预处理的背景图像

```bash
# 基本用法
python static_background_tracker.py video.mp4 -b background.png

# 交互模式
python static_background_tracker.py video.mp4 -b background.png -i
```

### 4.2 自动捕获背景

如果没有预处理背景，系统可以自动从视频捕获：

```bash
# 从视频开头捕获背景
python static_background_tracker.py video.mp4 -c

# 指定捕获帧范围
python static_background_tracker.py video.mp4 -c --bg-start 10 --bg-end 60
```

## 6. 常见问题

### Q1: 背景图像与视频分辨率不匹配？
**A**: 系统会自动调整背景图像大小，但建议预处理时保持一致。

### Q2: 光照变化导致检测不准确？
**A**: 
- 确保背景图像在相同光照条件下拍摄
- 调整 `--diff-threshold` 参数
- 考虑使用自适应阈值

### Q3: 背景中有固定物体（如水迷宫平台）？
**A**: 这些固定物体会被视为背景的一部分，不影响检测。
